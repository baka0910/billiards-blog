# 無料ブログプラットフォーム比較（ビリヤードブログ向け）

## おすすめ順

### 1. はてなブログ （最もおすすめ）
- **URL:** https://hatenablog.com
- **無料枠:** サブドメイン、画像300MB/月、3ブログまで
- **メリット:**
  - 日本語ネイティブのUI、始めやすい
  - 「はてなブックマーク」連携でバズりやすい
  - ブログ内コミュニティからの読者流入が見込める
  - Markdown対応、HTMLの埋め込みも比較的自由
- **デメリット:** 無料プランは広告表示あり、海外リーチは弱め
- **向いている人:** 日本のビリヤードファンに届けたい人

---

### 2. note
- **URL:** https://note.com
- **無料枠:** 完全無料、記事数・画像数の実質上限なし
- **メリット:**
  - UIがシンプルで書くことに集中できる
  - プラットフォーム内の発見性が高い（おすすめ表示など）
  - 有料記事の販売が可能（技術解説の収益化も）
  - スマホアプリからも投稿しやすい
- **デメリット:** カスタムHTML不可、デザインカスタマイズほぼ不可
- **向いている人:** 手軽に始めたい人、記事の収益化も視野に入れたい人

---

### 3. Blogger（Google）
- **URL:** https://www.blogger.com
- **無料枠:** 完全無料、15GB（Googleフォト共有）、広告なし、カスタムドメイン対応
- **メリット:**
  - 完全無料で広告が入らない
  - Googleアカウントだけで開始
  - HTMLテンプレートを完全カスタマイズ可能
  - 容量が実質的に大きい
- **デメリット:** テンプレートが古め、コミュニティ機能が弱い
- **向いている人:** 広告なしでHTMLを自由にカスタマイズしたい人

---

### 4. WordPress.com
- **URL:** https://wordpress.com
- **無料枠:** サブドメイン、1GBストレージ、基本テーマ利用可
- **メリット:**
  - 世界最大のブログプラットフォーム、情報が豊富
  - SEOに強く検索流入が期待できる
  - 将来的に有料プランで拡張可能
- **デメリット:** 無料プランはカスタムHTML制限あり、広告表示あり
- **向いている人:** 将来的に本格運用も見据えたい人

---

### 5. GitHub Pages
- **URL:** https://pages.github.com
- **無料枠:** 完全無料、1GB容量、月100GB帯域、SSL無料
- **メリット:**
  - 完全にカスタムHTML/CSS/JSが使える
  - 広告なし、高速（静的サイト）
  - 今ある HTML/CSS をそのままアップロード可能
- **デメリット:** Git/GitHubの知識が必要、コミュニティ流入なし
- **向いている人:** 技術に慣れている人、今あるHTMLファイルをそのまま公開したい人

---

### 6. FC2ブログ
- **URL:** https://blog.fc2.com
- **無料枠:** 10GBストレージ、テンプレート多数、カスタムHTML対応
- **メリット:**
  - 大容量ストレージ（10GB）
  - HTMLテンプレートの自由度が高い
  - アクセス解析機能あり
- **デメリット:** 1ヶ月未更新で大きな広告表示、サイトイメージがやや古い
- **向いている人:** 大容量が必要でカスタマイズしたい人

---

## まとめ比較表

| プラットフォーム | 費用 | 広告 | カスタムHTML | 日本語コミュニティ | 収益化 |
|:---|:---:|:---:|:---:|:---:|:---:|
| はてなブログ | 無料 | あり | 部分的 | ◎ | - |
| note | 無料 | なし | 不可 | ◎ | 有料記事可 |
| Blogger | 無料 | なし | 完全対応 | △ | AdSense可 |
| WordPress.com | 無料 | あり | 有料で可 | ○ | 有料プランで可 |
| GitHub Pages | 無料 | なし | 完全対応 | × | - |

## おすすめの選び方

- **とにかく手軽に始めたい** → note or はてなブログ
- **日本の読者を増やしたい** → はてなブログ
- **広告なしで自由にやりたい** → Blogger or GitHub Pages
- **今あるHTMLをそのまま公開** → GitHub Pages
